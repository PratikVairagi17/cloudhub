How to use this tool
1

Pick a Cloud

Click any cloud provider card above or use the tabs in the Checklist page to switch between GCP, AWS, Azure and Cloudflare.

2

Study the Reference

Open the Reference tab for deployment flows, key commands, and service cards for every service on that platform.

3

Select Services

In the Checklist tab, pick your primary deployment target then add-on services like databases, CDN, IAM and networking.

4

Generate Checklist

Click Generate to get a tailored, step-by-step deployment checklist with exact CLI commands for every step.

5

Deploy & Tick Off

Execute each step in your terminal and click to check it off. Watch your progress bar fill up to 100%.

6

Troubleshoot

If something breaks, use the troubleshoot section in Reference. Follow the systematic 7-step debug flow for any cloud issue.
